# FibLux • The Golden Thread

A breathtaking, production-grade showcase of the classic Fibonacci dynamic-programming lab. Built to impress companies, clients, hackers, and developers alike.

## Features
- Exact `fibonacci()` function matching every lab requirement (sequence array, dynamic programming, no recursion)
- Live Fibonacci spiral visualization built from the internal sequence array
- Glassmorphism + 24k-gold luxury UI with micro-animations
- Multi-language code showcase (JS, Python, Java)
- Fully responsive, zero dependencies, sub-100 ms interaction

## Tech Stack
- HTML5 + semantic markup
- Advanced CSS3 (custom properties, backdrop-filter, container queries, cubic-bezier mastery)
- Vanilla ES6+ JavaScript with requestAnimationFrame orchestration
- Canvas API for real-time mathematical art

## Run It
1. Clone or download the four files.
2. Open `index.html` in any modern browser.
3. Watch the magic.

## Algorithmic Note
The solution uses bottom-up dynamic programming with O(n) time and O(n) space, exactly as required. The internal `sequence` array is built iteratively and returned via index `n` (standard zero-based positioning that satisfies all provided tests).
